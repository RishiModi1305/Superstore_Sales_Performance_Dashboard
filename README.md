# Superstore_Sales_Performance_Dashboard
Sales performance analysis using Excel and Tableau on Superstore dataset

Analyzed 10,000+ sales records from the Superstore dataset to identify 
revenue drivers, profit leaks, and seasonal trends using Excel and Tableau.

## Tools Used
Excel (Pivot Tables, VLOOKUP, Conditional Formatting), Tableau Public

## Dataset
Superstore Sales Dataset — Kaggle

## Process
1. Cleaned 10,000-row dataset — removed duplicate/irrelevant columns, 
   fixed date formatting, added calculated Profit Margin column
2. Built 3 pivot tables in Excel for Region, Sub-Category, and 
   Monthly trends
3. Created an interactive Tableau dashboard with regional, 
   product-level, and time-series visualizations

## Key Findings
1. The West region contributed the highest revenue (~32% of total sales)
2. Tables and Bookcases sub-categories showed negative profit margins
3. Sales consistently peak in Q4 every year — useful for inventory planning
4. Technology category had the highest average profit margin
